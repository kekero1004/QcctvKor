QcctvKor 사용자 매뉴얼

1. 소개

QcctvKor는 QGIS에서 대한민국의 교통 CCTV를 실시간으로 모니터링할 수 있는 플러그인입니다. ITS 국가교통정보센터의 API를 활용하여 전국의 주요 도로 CCTV 영상을 제공합니다.

2. 설치 및 설정

  1. QGIS 플러그인 관리자에서 "QcctvKor" 검색
  2. 플러그인 설치
  3. ITS API 키 설정:
    • 플러그인 메뉴 → 설정
    • 발급받은 API 키 입력
    • API 키는 ITS 오픈 API에서 발급 가능

3. 주요 기능

3.1 CCTV 표시

  • 플러그인 메뉴 클릭 시 CCTV 위치가 지도에 표시됨
  • 각 CCTV 포인트에는 설치 위치명이 라벨로 표시됨

3.2 필터링

  • 지역별 필터링 (서울, 경기, 인천, 부산 등)
  • 도로 유형별 필터링 (고속도로, 국도, 시도)
  • CCTV 이름으로 검색 가능

3.3 영상 보기

  • CCTV 포인트 클릭 시 실시간 영상 팝업
  • 현재 시간 표시
  • 영상 캡처 기능

3.4 데이터 저장

  • 필터링된 결과를 CSV 파일로 저장
  • shapefile 형식으로 내보내기
  • CCTV 데이터 보고서 생성

4. 문제 해결

자주 발생하는 문제:

  • 영상이 표시되지 않는 경우:
    • 네트워크 연결 확인
    • API 키 유효성 확인
    • CCTV 정비/점검 여부 확인
  • 필터링이 작동하지 않는 경우:
    • 검색어 철자 확인
    • 플러그인 재시작 시도
